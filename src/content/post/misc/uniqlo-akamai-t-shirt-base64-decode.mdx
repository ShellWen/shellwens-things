其中 Base64 部分的内容为一个 shell 脚本，解码后内容如下：

```shell
#!/bin/bash

# Congratulations! You found the easter egg! ❤️
# おめでとうございます！隠されたサプライズを見つけました！❤️

# Define the text to animate
text="♥PEACE♥FOR♥ALL♥PEACE♥FOR♥ALL♥PEACE♥FOR♥ALL♥PEACE♥FOR♥ALL♥PEACE♥FOR♥ALL♥"

# Get terminal dimensions
cols=$(tput cols)
lines=$(tput lines)

# Calculate the length of the text
text_length=${#text}

# Hide the cursor
tput civis

# Trap CTRL+C to show the cursor before exiting
trap "tput cnorm; exit" SIGINT

# Set frequency scaling factor
freq=0.2

# Infinite loop for continuous animation
for (( t=0; ; t+=1 )); do
    # Extract one character at a time
    char="${text:t % text_length:1}"
    
    # Calculate the angle in radians
    angle=$(echo "($t) * $freq" | bc -l)

    # Calculate the sine of the angle
    sine_value=$(echo "s($angle)" | bc -l)

    # Calculate x position using the sine value
    x=$(echo "($cols / 2) + ($cols / 4) * $sine_value" | bc -l)
    x=$(printf "%.0f" "$x")

    # Ensure x is within terminal bounds
    if (( x < 0 )); then x=0; fi
    if (( x >= cols )); then x=$((cols - 1)); fi

    # Calculate color gradient between 12 (cyan) and 208 (orange)
    color_start=12
    color_end=208
    color_range=$((color_end - color_start))
    color=$((color_start + (color_range * t / lines) % color_range))

    # Print the character with 256-color support
    echo -ne "\033[38;5;${color}m"$(tput cup $t $x)"$char\033[0m"

    # Line feed to move downward
    echo ""

done
```

# 背景

优衣库在 2025 年 6 月 20 日，为其「PEACE FOR ALL」慈善 T 恤项目推出了与云服务及网络安全公司 Akamai 合作的全新设计 T 恤。这是继之前成功合作后，Akamai 再次作为合作伙伴参与该项目。

<AppPicture src={akamaiTShirtBack} alt="T 恤背面" />

这次合作的设计充满了对互联网发展史的致敬：

- **米色设计**：T 恤的浅褐色是对早期「米色盒子」电脑塑料外壳的怀旧，那是互联网黎明期的象征
- **胸前心形**：象征着互联网连接人们，将世界引向更美好未来的力量
- **背面代码**：最特别的是 T 恤背面印着一段**真实的代码**，这段代码引用了支撑互联网的开源操作系统 Linux，象征着开放、共享与连接的理念

正如 Akamai 所表达的：「世界中で愛されるこれらの共通言語の力を借りて、Akamai が、世界を導くブランドとその先のユーザーとを結びつけ、より安全で、よりつながる未来を共に築いていきたい」——希望借助这些被全世界所喜爱的共通语言的力量，让 Akamai 能够连接世界领先品牌及其用户，共同构建一个更安全、更紧密相连的未来。

所有「PEACE FOR ALL」T 恤的销售利润都将捐赠给联合国难民署（UNHCR）、救助儿童会（Save the Children）、国际计划组织（Plan International）等国际人道主义组织，用于支持全球各地的难民和处境困难的儿童。

而这篇文章，就将聚焦于 T 恤背面的那段神秘代码——让我们一起来看看这段 Base64 编码背后隐藏着什么。
